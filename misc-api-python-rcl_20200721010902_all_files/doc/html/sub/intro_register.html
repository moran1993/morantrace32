Registers¶

Register access is provded by the RegisterService class. Each connection has each own instance, which can be used through it’s Register attribute.

Reading Registers¶

>>> reg = dbg.register.set(name='R0')
>>> print(reg)

Writing Registers¶

>>> dbg.register.write(name='R0', value=0)
